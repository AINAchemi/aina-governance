# AINA: AI-Native Governance Architecture

AINA는 AI 거버넌스 구조, 자율성 상한선 분류 체계, 인간 책임 흐름 모델을 제안하는 독립 연구 프레임워크입니다.

## 현재 상태
- 버전: v0.1 (개념 프레임워크 제안 단계)
- 구현, 실험, 정책 적용 없음

## 저장소 구조
- docs/: 연구 스펙 문서 및 프로토콜 초안
- archive/: 초기 사고 기록 (추가 예정)

## 연구 주제
- AI 거버넌스 아키텍처
- 인간 의사결정 개입 구조
- AI 자율성 분류 체계
- AI 협업을 위한 인지 상태 직렬화
